# 西遊記(NISHIYUKI) マニュアル

09班

空を飛ぶ孫悟空を操作し、ハイスコアを目指すアクションゲーム。障害物を避けてより高得点を目指そう。  
動作環境:PC 開発環境:Greenfoot

# 操作方法
- 上キー：上昇
- 下キー：降下

# ルール
孫悟空は自動で進むので、右から障害物(岩と龍)がどんどん流れてくる。高度を調整して障害物をかわす。岩や龍にぶつかるとゲームオーバー。より長い時間飛ぶと高得点。
一定時間で流れてくるバナナに触れると、孫悟空が巨大化しスーパー悟空にパワーアップする。岩に触れてもゲームオーバーにならず破壊できる。ただし龍に触れると変わらずゲームオーバーになる。

# キャラクター
![悟空](./Goku.png)
- 孫悟空(まごごくう)  
プレイヤーが操作するキャラクター。頭に輪っかを付けた猿。何か悪いことをして逃げてるらしい。バナナを取るとスーパー悟空になる。

![岩](./ozyama_storn.png)
- お邪魔  
障害物。ただの岩。これに当たるとゲームオーバーになる。画面の上下に出現する。スーパー悟空の時に触れると破壊できる。

![龍](./Ryu.png)
- 龍  
障害物その2。龍。これに当たるとゲームオーバーになる。一定周期で出現し、高速で飛んでくる。スーパー悟空で触れてもゲームオーバーになる。

![バナナ](./fruit_banana.png)
- バナナ  
バナナ。一定周期で出現する。孫悟空が取得するとスーパー悟空になる。スーパー悟空は一定時間経過で元に戻る。  

各キャラクター、及び背景にはいらすとやのイラストを使用しています。タイトルロゴの作成には、ChatGPTを使用しています。
